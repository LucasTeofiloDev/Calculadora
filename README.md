
## Calculadora JS
Projeto desenvolvido para consolidar fundamentos de lógica de programação, manipulação dinâmica de elementos via DOM e gerenciamento de estados, utilizando sintaxe moderna de JavaScript.

# Funcionalidades
O sistema simula uma calculadora funcional, permitindo operações aritméticas básicas (soma, subtração, multiplicação, divisão e resto). Possui tratamento para números decimais, limpeza de memória (Reset) e uma interface estruturada para proporcionar uma experiência de usuário clara e responsiva.

# Tecnologias
JavaScript (ES6+): Lógica de cálculo e manipulação de eventos com foco em Arrow Functions.

HTML5: Estruturação semântica dos componentes da calculadora.

CSS3: Estilização e layout do visor e botões.

# Conceitos Aplicados
A implementação foca na captura de eventos através de Arrow Functions dentro de métodos como forEach e addEventListener, garantindo um código mais conciso e moderno. Além disso, utiliza gerenciamento de variáveis globais para controle de estado, conversão de tipos de dados (parseFloat e toString) e estruturas de controle condicional para processamento das operações matemáticas.
